# Avremu -- Avr Emulator
C'est un simple émulateur qui ne supporte que les Atmega328p pour l'instant et qui est écrit en C.

Le but est à terme de l'adapter à un microcontrôleur comme le Raspberry Pi Pico ou l'ESP32 afin de faire tourner du code Arduino Uno de façon dynamique dessus.
